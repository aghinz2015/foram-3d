# Foram 3D

## Setup

Install NodeJS:

```bash
  $ brew install node
```

Clone repository:

```bash
  $ git clone git@github.com:aghinz2015/foram-3d.git
  $ cd foram-3d
```

Install dependencies:

```bash
  $ npm install
```

Install Bower:

```bash
npm install -g bower
```

Install Bower dependencies:

```
bower install
```

Run gulp:

```bash
  $ gulp
```
